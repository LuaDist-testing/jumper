Module init

Init file.

This file was meant for self-containment purposes. On some Lua distributions, the package of modules contains this explicit pattern. In this case, you can simply write the following: 

     local Jumper = require ("Jumper")
Anyway, the default method requires to call explicitly init.lua 
     local Jumper = require ("Jumper.init")
Optionnally, you can also add the pattern init.lua in your package of modules. Therefore, you can shorten the previous syntax 
     package.path = package.path .. (";.\\?\\init.lua")
